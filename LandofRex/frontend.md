# 프론트엔드 포팅 매뉴얼

- **프레임워크**: React
- **빌드 도구**: Vite
- **주요 의존성**: `react`, `react-dom`, `axios`, `zustand`, `sweetalert2`
- **주요 개발 의존성**: `vite`, `eslint`, `@vitejs/plugin-react`

## 1. 프로젝트 의존성 설치

- Node.js 설치 ( v20.15.1 )
    - 설치 확인: node -v
- 프로젝트 의존성 설치
    - npm install

## 2. 개발 환경에서 실행

- npm run dev
